# 🌍 Travel Blog Website

A responsive and dynamic **Travel Blog Web Application** where users can explore destinations, read travel stories, and share experiences.

---

## ✨ Features

- 🌐 Beautiful and responsive UI  
- 🗺️ Explore travel destinations  
- 📝 Blog posting / travel stories  
- 🔍 Search and browse blogs  
- ⚡ With payment gateway 

---

## 🛠️ Tech Stack

**Frontend:**
- React Js
- Tailwind 

**Backend:**
- Node.js / Express  

**Database:**
- MySQL  

---

## Installation

### 1. Clone the Repository

```bash
git clone <repo_url>
```

### 2. Navigate to the Project Directory

For the Frontend:

```bash
cd frontend
```

For the Backend:

```bash
cd backend
```
### 3. Install Dependencies

```bash
npm install
```

### 4. Run the Development Server

```bash
npm run dev
```

## Notes

- Replace `<repo_url>` with the actual Git repository URL.
- Run the above steps separately for both the frontend and backend folders if they are maintained as separate projects.
- Ensure that all required environment variables are configured before starting the application.
